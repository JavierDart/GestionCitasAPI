##Gestion de Citas API
Esta API ha sido desarrollada con .NET 8 con el objetivo de proporcionar un sistema backend robusto para la gestión de citas en un estudio de tatuajes. Su finalidad es centralizar y facilitar la administración de reservas, permitiendo a clientes y al tatuador organizar de forma eficiente su agenda.
La API expone una serie de endpoints REST que permiten realizar operaciones completas sobre las citas, incluyendo la creación, consulta, actualización y eliminación de registros. Cada cita contiene información relevante como el cliente, la fecha y hora, y la descripción del servicio solicitado.
Además, está diseñada siguiendo buenas prácticas de desarrollo, lo que permite su integración con diferentes tipos de clientes, como aplicaciones móviles (por ejemplo, una app en .NET MAUI) o aplicaciones web. Esto asegura flexibilidad y escalabilidad, facilitando futuras ampliaciones como autenticación de usuarios, gestión de imágenes de tatuajes o notificaciones.
Entre sus principales objetivos destacan:

✅ Digitalizar la gestión de citas del tatuador
✅ Reducir errores y solapamientos en la agenda
✅ Permitir a los clientes solicitar citas de forma sencilla
✅ Servir como backend para aplicaciones móviles y web

En resumen, esta API constituye la base tecnológica para una aplicación moderna de gestión de estudio de tatuajes, preparada para evolucionar hacia un sistema más completo y profesional.
